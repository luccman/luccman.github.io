Project Title: LIRA Academic Project

Short Description

In 2022, my classmates and I developed an academic project called LIRA, a computer application designed for a university system. The app was built using VB.NET in Visual Studio 2019 and connected to Microsoft Access for data management. It featured a user-friendly interface with drag-and-drop functionality and double-click actions to manage class names, schedules, and more.

Tech Stack Used

  • VB.NET – For application development.
  • Visual Studio 2019 – As the IDE for building the project.
  • Microsoft Access – For database management.

Features / Highlights

  • User-friendly interface with drag-and-drop and double-click functionality.
  • Integration with Microsoft Access for managing class data, schedules, and student information.
  • Customizable forms for adding, editing, and deleting records.
  • Real-time data updates and validation.

Your Role / What You Did

As part of the team, I contributed to the following:

  • Designed and implemented the user interface using VB.NET's drag-and-drop tools.
  • Connected the application to Microsoft Access for seamless data management.
  • Developed key features such as form validation and real-time updates.
  • Collaborated with teammates to debug and optimize the application.
